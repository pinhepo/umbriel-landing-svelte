## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/umbriel-landing-svelte.git
cd umbriel-landing-svelte
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Deployment

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Visit [Vercel](https://vercel.com) and sign up/login with GitHub
3. Click "New Project"
4. Import your repository
5. Select "Svelte" as the framework
6. Click "Deploy"

Vercel will automatically detect your Svelte configuration and deploy your site.

### Alternative Deployment Options

You can also deploy to:
- Netlify (Free)
- GitHub Pages (Free)
- Cloudflare Pages (Free)
